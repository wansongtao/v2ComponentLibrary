# v2component
个人组件库，包含常用js、常用css、常用element-ui的组件二次封装、axios封装、router封装、自定义组件等等。
主要技术：vue2全家桶+element-ui+axios+eslint。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
